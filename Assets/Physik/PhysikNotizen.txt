		Notizen: 
		
		1.	Physikalisches Modell für den Federdämpfer:
			Basiert auf dem Hooke's Gesetz: 
				F_feder = -k * x; 
					wobei: 
						k = Federkonstante 
						x = Auslenkung der Feder
			
			und dem Dämpfungsgesetz: 
				F_dämpfung = -c * v; 
					wobei: 
						c = Dämpfungskonstante 
						v = relative Geschwindigkeit 
				
				Fragen: 
					1. 	In welchem verhältnis stehen die Werte jeweils zueinander? 
						Antwort: 
							1. Verhältnis Federkonstante k und Auslenkung x
								Die Federkraft ist direkt Proportional zur Auslenkung der Feder 
									-> ist die Auslenkung größer, so ist es auch die Federkraft 
								Ist die Federkonstante k größer, so ist diese Steifer
									->	D.h bei einer gegenen Auslenkung x wird eine größere Federkraft erzeugt, welche das Fahrzeug schneller wieder in seine Ausgangslage bringt 
							
								1.1 Wieso das wichtig für das Fahrverhalten ist: 
									Eine steifere Fäder(hohes k) führt zu einem härteren Fahrwerk
										->	Fahrzeug reagiert schneller auf Auslenkungen
											->	Fahrkonfort leidet aber darunter
									Ist die Feder weicher
										->	höher Fahrkonfort 
											->	geringere Fahrsicherheit 
								
										
							2.	Verhältnis Federkonstante c und relative geschwindigkeit: 
								2.1	Ebenfalls linear
										-> umso höher die dämpfung c ist, umso schnell nimmtt die relative geschwindigkeit ab 
											->	umso schneller kommt das system zum stehen 
											Das passiert weil: 
												-umso gedämpfter das System ist, umso schneller nimmt die Energigie ab
								
								2.2	warum das wichtig für das Fahrverhalten ist: 
										Umso gedämpfter ein System ist umso weniger kann das Fahrzeug schwingen 
											->	Schlechter Fahrkonfort
											->	Stabileres Fahrverhalten
										-Umso geringer ein System gedämpft ist, umso mehr kann es schwingen
											-> Besserer Fahrkonfort 
											-> Schwammigeres Fahrverhalten 
												->	Geringere Fahrsicherheit 
							
							
							3.	Verhältnis Federkraft und Dämpfungskraft:
								3.1	Verhältnis: 
									->	Das dynamische Verhalten des Fahrzeuges wird durch das verhältnis zwischen Federkraft und Dämpfungskraft bestimmt
								
								3.2	Kritische Dämpfung:
									->	Kritische Dämpfung ist erreicht:
										->	wenn c so gewählt ist, dass das System keine schwingenden bewegungen mehr macht
											-> Stattdessen soll das system das so schnell machen, dass es noch in seine Gleichgewichtslage(auch ruhe position genannt) bringt
										->	Das ist ein häufig gewolltest verhalten 
												->	Das liegt daran, dass sich das Fahrzeug so ohne übermäßiges Schwingen schnelle stabilisierung ermöglicht 	
										
								3.3	->	Überdämpfung:
										->	Ist C(Dämpfungskonstante) zu groß:
											-> Dann bewgt sich das System langsam in seine Gleichgewichtslage zurück
												->	Dabei schwingt es nicht
												->	Es kommt zu einem trägen Fahrgefühl 
								3.4	->	Unterdämpfung:
										->	Ist C zu klein:
											->	Dann schwingt das Fahrzeug mehr
											->	Es kommt zu einem Schwammigen Fahrgefühl 
											
							4.	Warum das wichtig für das Fahrverhalten ist: 
								4.1	Fahrstabilität: 
									->	Umso abgestimmter die Federkraft und Dämpfungskraft sind:
										->	Umso stabiler, konfortabler und reaktionsfreudiger ist das Fahrgefühl 
									->	Sind die beiden kräfte aber nicht gut aufeinander abgestimmt:
										->	Dann entsteht ein unangenehmes, instabilles und unsicheres Fahrverhalten

													Dämpfungskonstante c:

															Weil:

				Fragen: 
					1. 	Inwiefern unterscheiden sich die Eigenschaften von Rad und Kettenfahrzeugen? 
						Antwort: 
							1.1	Beweglichkeit und wendigkeit 
								->	Radfahrzeuge:
									-	beweglicher
										->	besonders auf befestigten Straßen
									-	Erreichen höhere Geschwindigkeiten
									-	Geringerer Rollwiderstand führt zu mehr Effizienz:
										->	da geringerer Treibstoff verbrauch
									-	Schlechter für unwegsames Gelände zugebrauchen 
										->	das liegt daran, dass reifen leichter in weichen Boden stecken bleiben 
								->	Kettenfahrzeuge: 
									-	Gelände tauglicher
										->	das liegt daran, dass das Gewicht auf eine größere Fläche verteilt ist 
									-	Sind auf festen Untergründen meist langsamer
									-	Können besser in engen oder unwegsamen Gelände Manövrieren
										->	das liegt daran, dass sie sich an Ort und Stelle drehen können 
							
							1.2	Aufhängung und Federung: 
								->	Radfahrzeuge:
									-	Höherer Fahrkonfort
										->	Individuelle Federung und Aufhängung pro Rad 
									-	Höhere Komplexität(
										->	durch z.B. Individuelle Federung und Aufhängung 
								->	Kettenfahrzeuge:
									-	Robustere und einfachere Federung 	//=> Robuster meint in dem Fall:  Die Kette ist widerstandsfähigker und dadurch langlebiger 
										->	Die Fahrzeugkette ist mit dem Gesammten Fahrzeug verbunden, wodurch sie weniger komplex ist 
									-	Federung ist weniger Anpasstbar 
										->	Weniger Fahrkonfort
										->	aber besser auf unwegsamen Gelände 
							
							1.3	Einfluss auf das Fahrverhalten: 
								->	Radfahrzeug: 
									-	Bessere Fahrbarkeit auf Straßen
										->	Das liegt an der bessere Manövrierfertigkeit und individuellen Federung 
									-	Schlechter im Gelände 
										-> durch z.B. im Gelände feststeckende Reifen
								->	Kettenfahrzeuge:
									-	Besser im Gelände 
										->	langelebige Kette
										->	Verteilung des Gewichtes auf größere Fläche
									
					2. 	Welchen einfluss hat die Fahrwerkshöhe auf das Fahrverhalten? 
						Antwort:
							2.1	Schwerpunkt und stabillität: 
								-> Höhergelegtes Fahrwerk: 
									-	Erhöhter Schwerpunkt
										->	Erhöhtes risiko des umkippens in Kurven oder bei schnellen bewegungen 
									-Verbesserte Bodenfreiheit
										->	Fahrzeug rumpf ist höher wodurch mehr hindernisse überfahren werden ohne das Fahrzeug zubeschädigen 
									-	Verschlechterte Kurvenlage
										->	Durch den höheren schwerpunkt des Fahrzeuges nimmt die Neigung des Fahrzeuges in Kurven zu
								->	Tiefergelegtes Fahrwerk: 
									-	Höhere stabilität bei Kurven und höheren Geschwindigkeiten
										->	Das liegt am niedrigeren Schwerpunkt 
									-	verschlechterte Bodenfreiheit
										-> niedirigere Hindernisse können nur überfahren werden(wenn überhaupt)
											-> durch den niedrigeren Schwerpunkt

		3.	Vereinfachungen: 
			3.1	Vereinfachungen der Feder und Dämpferberechnung: 
				-Um die berechnung zu vereinfachen kann man folgende Dinge in betracht ziehen: 
					-Federkraft hängt liniear von der Auslänkung ab
					-Dämpfkraft linear von der Geschwindigkeit
				
				Begründung:  
					-Performanter
					-Realistisch genug 
					-einfacher zu implementieren 
			
			3.2	Ausblenden von nicht Essentiellen Details: 
					-Es ist nicht nötig alles Physikalisch genau zu halten
						-kleinere Unebenheiten z.B. könnten Ignoriert werden 
						
					Begründung: 
						um die implementation zuvereinfachen 
						
				Fragen: 
					1.	Wie könnte eine Gleichung aussehen, wie sie in 3.1 beschrieben ist? 
						Antwort: 
							Die Vereinfachte Gleichung von der Federkraft ist: 
								F_feder = -k * x;
									->	k = Federkonstante, x = Auslenkung der Feder
							Die Vereinfachte Gleichung der von der Dämpfungskraft ist: 
								F_dämpfung = -c * v; 
									->	c = Dämpfungskonstante, v = relative Geschwindigkeit
							Daraus ergibt sich: 
								F_gesamt = F_feder + F_dämpfung = -k*x-c*v; // => -c*v weil das + aufgrund c's vorzeichen zu minus wird 
								
					2. 	Inwiefern beeinflusst das ignorieren kleinerer unebenheiten die implementation? 
						2.1	Welche vorteile und welche nachteile gäbe es?
							->	Vorteile:
								-	Verbesserte Performance 
									->	das system muss nicht auf jeden kleinen Hubel reagieren
								-	Einfachere Implementation 
									-> durch weniger faktoren die zu berüchtigen sind, wird die Implementationszeit reduziert 
								-	Stabileres verhaltem 
									->	dadurch schwankt die Simulation weniger, was zu weniger störendem verhalten führen kann 
							->	Nachteile: 
								-	Redzierter Realismus 
									-> verlust an Detailgenauigkeit
										->	Könnte die Spielerfahrung in realistischen Rennsimmulationen z.B. beinträchtigen. 
										->	Das ist mir aber egal.
								
							=> Sollte man machen. 
								
					3.	Was könnten weitere Vereinfachungen sein? 
						3.1 Vernachlässigung  von Roll- und Luftwiderstand: 
							-	der Fokus auf Roll- und Luftwiderstand kann reduziert werden, da diese Kräfte häufig kleiner sind als 
								die Feder- und Dämpferkräfte
									-> somit nehmen sie weniger Einfluss auf das Fahrverhalten
								Vorteil:
									->	Geringerer Rechenaufwand 
									->	Einfachere Implementation 
								Nachteil: 
									->	Weniger realistische Fahrphysik // => Fragt sich halt nur: wie viel weniger realistisch man. 
								
								=> ich denke ich werde es verwenden, werde bei der Implatation nochmal abwiegen wollen. 
							
						3.2	Approximative Reibungsmodelle 
							-	Anstatt komplexer Reibungsmodelle könnte ein einfaches Coulomb - Reibungsmodell verwendet werden 
								
								++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
								+################################################################################################+
								+################################################################################################+
								+##   Coulomb Reibungsmodell: 																   ##+	
								+##		->	Gibt an wie viel Kraft benötigt wird, um ein Objekt zu bewegen.					   ##+	
								+##			->	Haftreibung:																   ##+
								+##					-	Es muss eine gewisse kraft überwunden werden, um ein Objekt zu bewegen.##+			
								+##			->	Gleitreibung																   ##+				
								+##					-	wenn sich ein Objekt bewegt, dann bleibt die Reibungskraft konstant    ##+
								+##						-	ist für gewöhnlich kleiner als die Haftreibung                     ##+
								+################################################################################################+
								+################################################################################################+
							    ++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
								
								Vorteil: 
									-	weniger berechnungen pro Zeitschritt
								Nachteil:
									-	Bei extremen Fahrmernövern könnten Unrealistisches Verhalten auftreten 
						
						3.3 Vereinfachte Kontaktpunkte: 
							-	Man könnte doe Kontaktpunkte vereinfacht als einzelnen punkt oder linie behandeln 
								Vorteil: 
									->	Reduzierte komplexität der berechnung 
								Nachteil: 
									->	Ungenau bei uneben flächen 
									->	Ungenau bein Seitlichen kräften
